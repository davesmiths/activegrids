What it does is take this:

	...
	<div class="col">
</div>

and make it

<div class="ags ags-gt1 ags-gt2 ags-gt3 ags-4 ags-lt5 ags-lt6 ags-lt7 ags-lt8 ags-lt9 ags-lt10 ags-lt11 ags-lt12">
	<div class="col">
	...
	<div class="col">
</div>

Options

find
	CSS selector
	Defaults to ".ags"

width
	number of pixels
	Default to 60

minWidth
	true or false
	Defaults to false
	When true the column must always be larger than the width set
	When false the column must always be smaller than the width set

customClass
	Text
	Defaults to "ags"
	A bit like a namespace, all the classes are prepended with this

jquery
	URL of the jQuery library
	defaults to "//ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"
	Used in the polyfill for browsers that didn't have document.querySelectorAll or document.addEventListener
